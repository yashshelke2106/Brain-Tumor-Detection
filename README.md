# Brain Tumor Detection

This project trains a convolutional neural network to classify brain MRI images and identify tumor types.

## Contents
- `app.py` – Flask app for inference
- `Brain_Tumor_CNN_Project.ipynb` – training notebook
- `requirements.txt` – Python dependencies
- `setup_and_run.bat` – Windows helper script

## Notes
Large model artifacts and training datasets are intentionally ignored in this repository to keep the Git history lightweight and compatible with GitHub limits.
